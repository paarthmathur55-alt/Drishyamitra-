# DrishyaMitra – AI Photo Management System

DrishyaMitra is an intelligent photo management system that uses Artificial Intelligence to organize, manage, and retrieve images efficiently. It helps users automatically categorize photos, search images using smart filters, and manage large photo collections easily.

## 🚀 Features

* 📸 Upload and manage photos
* 🤖 AI-based image organization
* 🔍 Smart image search and filtering
* 🗂 Automatic categorization
* ☁ Secure photo storage
* ⚡ Fast and user-friendly interface

## 🛠 Tech Stack

**Frontend**

* React
* TypeScript
* Vite
* Tailwind CSS

**Backend**

* Node.js
* Express.js

**Storage**

* Local file storage / uploads folder

**AI Integration**

* Image recognition and classification models

## ⚙ Installation

Clone the repository:

```
git clone https://github.com/mayankkishanmathur-coder/DrishyaMitra.git
```

Navigate to the project directory:

```
cd DrishyaMitra
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

The application will start on:

```
http://localhost:5173
```
